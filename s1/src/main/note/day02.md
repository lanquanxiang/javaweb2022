# 3. JSP组成
1. JSP代码块 <%  Java代码   %>
2. JSP表达式 <%=表达式(不要加分号) %>
3. JSP全局（静态）代码或变量  <%!   %>
4. JSP指令
	1. <%@page %> 配置页面相关内容
	2. <%@include %> 包含其他页面
	3. <%@taglib %> 配置第三方库（标签）
